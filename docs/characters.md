# 캐릭터 정의서 — 「사건파일 503호」 NPC 6인

> 아트 바이블(14) 규칙 적용: 인물당 **실루엣만으로 식별되는 시그니처 외형 1개** 필수 + 형태 언어(둥근=아군, 각진=상사) + 도트 정면 상반신(64×64 다운스케일 설계) + 표정 3종(기본/미소/당황·붕괴).
> 도트화는 후처리(팔레트 통일+NN 다운스케일)라 생성은 상반신 클로즈업 512×512로 뽑고 내린다.

## 공통 스타일 키워드 (모든 인물에 적용)
- 도트 감성: 90년대 PC 패키지 게임식 픽셀 포트레이트, 제한 팔레트(32색 이내), 굵은 아웃라인 없이 명암으로 형태, 배경 단색(크로마 제거용)
- 톤: 가우스전자식 생활 캐릭터 — 과장된 시그니처, 그로테스크하지만 매력적(Golden Idol 원칙)

## 구본식 (52) 영업부장 — 사건1 진범
- **시그니처**: 뿔테안경 + 헤어존 확장(벗겨지기 시작) + 넥타이 느슨
- 체형: 각진 어깨, 살짝 나온 배 (허세 체형)
- 표정: 기본=턱 치켜든 허세 / 미소=광대 올라간 영업 미소 / 붕괴=안경 들어 올리며 식은땀
- 컬러: 감색 정장 + 붉은 계열 넥타이(상사 상징)

## 차민재 (34) 마케팅 대리 — 활화산 실무자
- **시그니처**: 짙은 다크서클 + 야구점퍼형 사복 재킷 + 이어폰 한쪽
- 체형: 앞으로 기운 어깨(피로 체형)
- 표정: 기본=미간 주름(욱 참는 중) / 미소=한쪽 입꼬리 비웃음 / 붕괴=머리 쥐어뜯는 제스처
- 컬러: 회색 재킷 + 검정 티

## 이상록 (27) 신입 — 팩폭 상식주의
- **시그니처**: 두툼한 눈썹 + 반듯하게 치켜올린 이마 라인 + 새 와이셔츠(다림질 선명)
- 체형: 꼿꼿한 정면 (눈치 제로)
- 표정: 기본=정면 응시 무표정 / 미소=어색한 사회 초년생 미소 / 붕괴=입이 살짝 벌어진 당황
- 컬러: 흰 와이셔츠 + 네이비

## 전순덕 (45) 경리과장 — 철벽 원칙주의
- **시그니처**: 낮은 번 헤어 + 서류봉투를 가슴에 댄 자세 + 얇은 금테 안경
- 체형: 단단한 정면, 어깨선 수평 (철벽 체형)
- 표정: 기본=입술 일자 굳은 무표정 / 미소=눈만 살짝 웃는 제한 미소 / 붕괴=안경 너머로 눈 크게 뜸
- 컬러: 차콜 블레이저 + 브로치

## 마루팡 (38) 총무 과장 — 능글 월루
- **시그니처**: 커다란 사원증 목줄 + 팔짱 낀 느슨한 자세 + 반쯤 올라간 입꼬리
- 체형: 뒤로 기운 느긋 체형
- 표정: 기본=능글 미소 / 미소=이가 보이는 활짝 / 붕괴=미소가 굳은 채 눈만 흔들림
- 컬러: 라벤더 셔츠 + 목줄 파랑

## 오복자 (61) 청소 담당 — 전지적 목격자
- **시그니처**: 고무장갑 한쪽 + 걸레 어깨에 걸침 + 깊은 팔자주름
- 체형: 자연스럽게 굽은 등 (연륜 체형)
- 표정: 기본=눈가 주름 가득한 미소 / 미소=이 없는 활짝 / 붕괴=눈썹 치켜올린 "글쎄요~"
- 컬러: 남색 작업복 + 고무장갑 노랑
